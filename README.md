# ⚡️Skkuduler
안녕하세요
저희 소프트웨어공학개론 3팀은 학사일정을 효율적으로 관리할 수 있는 모바일 웹서비스인 Skkuduler를 만들었습니다.
Skkuduler를 통해 효율적으로 학사일정을 관리해 보세요!
